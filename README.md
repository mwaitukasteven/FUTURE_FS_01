## 🌟 Features

- ✅ **Fully Responsive** - Works perfectly on all devices (mobile, tablet, desktop)
- ✅ **Dark/Light Mode** - Toggle between themes with smooth transition
- ✅ **Interactive UI** - Smooth animations, hover effects, and typing animation
- ✅ **Project Showcase** - Display projects with tech stack and GitHub links
- ✅ **Skills Section** - Visual representation of technical skills
- ✅ **Experience Timeline** - Professional work experience display
- ✅ **SEO Friendly** - Optimized for search engines

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- Font Awesome Icons

- GitHub Pages (Frontend)
